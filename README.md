## Overview
Программа для обработки данных о владельцах автомобилей. Читает информацию 
из входного файла в формате CSV (разделитель ";"), отбирает владельцев 
автомобилей марки "ВАЗ" и сохраняет результаты в выходной файл.

## Author
Stanislavv-Kabanov

## Usage
1. Скомпилировать программу:
   make

2. Запустить программу:
   make run
   или
   ./bin/military_program

3. Входной файл: data/input.txt (формат CSV с разделителем ";")

4. Выходной файл: data/output.txt (отфильтрованные записи)

5. Очистка:
   make clean

6. Запуск тестов:
   make test
